# Ultimate Weather App 🌦️

A simple and modern weather application that allows users to check real-time weather conditions for cities around the world.
The application uses the **OpenWeather API** to fetch weather data and displays it with a clean and responsive interface.

This project was created to practice **API integration, front-end development, and browser features such as geolocation and voice recognition**.

---

## Features

* Search weather by **city name**
* Supports **cities worldwide**
* **Current location weather** using browser geolocation
* **5-day weather forecast**
* **Temperature chart visualization**
* Weather details including:

  * Temperature
  * Humidity
  * Wind speed
  * Pressure
  * Visibility
  * Feels like temperature
  * Sunrise and sunset time
* **Voice search** for quick city lookup
* **Search history** stored in local storage
* Responsive design for both desktop and mobile devices
